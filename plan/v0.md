# V0 Implementation Steps

1. Create a Vite + React + TypeScript app.
2. Configure the project for fully static output.
3. Add a GitHub Pages-compatible Vite `base` configuration.
4. Add base CSS files:
   - `src/styles/base.css`
   - `src/styles/layout.css`
   - `src/styles/components.css`
   - `src/styles/themes.css`
5. Add the initial app shell:
   - top bar region
   - editor region
   - optional virtual input panel region
6. Add basic light and dark theme variables.
7. Add a minimal `TopBar` component.
8. Add a minimal `TextEditor` component with a native `<textarea>`.
9. Add accessible labels for the editor and top bar controls.
10. Confirm `npm run dev` starts successfully.
11. Confirm `npm run build` completes successfully.
12. Confirm `npm run preview` serves the production build.
