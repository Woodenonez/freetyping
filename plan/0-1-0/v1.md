# V1 Implementation Steps

1. Create the editor adapter:
   - `getValue()`
   - `setValue(value)`
   - `getSelection()`
   - `setSelection(start, end)`
   - `focus()`
   - `insertText(text)`
2. Add centralized text insertion helper logic.
3. Use `textarea.setRangeText()` for adapter-driven insertion where possible.
4. Dispatch a normal `input` event after adapter-driven insertion.
5. Keep the DOM textarea as the source of truth for committed editor text.
6. Add clear text action.
7. Add copy text action.
8. Add opt-in local text persistence with `localStorage`.
9. Add debounced text persistence.
10. Add clear saved text action.
11. Add persistent settings for:
    - selected input mode
    - virtual panel visibility
    - theme
12. Add input mode types.
13. Add an input manager.
14. Add system input mode.
15. Add English direct input mode.
16. Add Chinese Pinyin placeholder input mode.
17. Add input mode selector UI.
18. Route editor events through the input manager:
    - `compositionstart`
    - `compositionupdate`
    - `compositionend`
    - `beforeinput`
    - `input`
    - `keydown`
19. Track native composition state.
20. Block web input transformations while native composition is active.
21. Ensure system input mode never transforms text.
22. Add QWERTY keyboard layout data.
23. Render the virtual keyboard.
24. Track physical `keydown` and `keyup` with `event.code`.
25. Highlight active virtual keys.
26. Add virtual key click insertion through the editor adapter.
27. Prevent textarea blur on virtual key mouse down.
28. Return focus to the textarea after virtual key insertion.
29. Add virtual mouse model.
30. Render left, middle, and right virtual mouse regions.
31. Track mouse clicks inside the app area.
32. Highlight virtual mouse buttons during interaction.
33. Avoid global context-menu blocking.
34. Verify native typing, selection, paste, undo, and redo.
35. Verify native system IME composition does not duplicate text.
36. Verify selected input mode persists after refresh.
37. Verify local text restores only when local saving is enabled.
38. Run `npm run build`.
